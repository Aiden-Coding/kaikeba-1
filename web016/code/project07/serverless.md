## 什么是serverless

单体应用    Nginx+代码+DB+..  一个服务器
    Nginx  代码(拆分代码的需求非常强烈)  DB   多个服务器
    mono 单仓库
微服务  账户,交易,商品, 多功能服务器拆分
    稳定性,隔离性,可维护性
    网关  怎么拆分?   架构成本特别高
serverless Nginx Function 基础设施
